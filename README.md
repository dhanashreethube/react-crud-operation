# React CRUD Operation

This is a simple React CRUD (Create, Read, Update, Delete) application.

## Tech Stack
- React
- React Router DOM
- React Hook Form
- Zod validation
- Parcel bundler
- Mock API (mockapi.io)

## Features
- Create new user
- View user list
- Edit user details
- Delete user

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. App runs on `http://localhost:1234`

## How to Add New Fields to the Form
1. Add field config in `src/config/userFormFields.js`
2. Add validation in `src/schema/userSchema.js`
3. Form UI updates automatically

## Mock API
Mock API is created using **mockapi.io**
